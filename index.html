HTML: Wichtige Elemente

Struktur

  • <html>, <head>, <body> – Grundgerüst
  • <div>, <span> – Container
  • <header>, <main>, <footer> – Semantische Struktur

Texte

  • <h1>...</h1> – Überschriften
  • <p> – Absatz
  • <strong>, <em> – Hervorhebungen
  • <a href="..."> – Links

Formulare

  • <input type="text">, <input type="password">, <input type="checkbox">
  • <textarea> – Mehrzeiliges Eingabefeld
  • <select><option> – Dropdown
  • <button> – Button

Medien

  • <img src="bild.jpg"> – Bild
  • <audio controls> – Audio
  • <video controls> – Video
  • <iframe> – Externe Inhalte

CSS: Wichtige Eigenschaften

  • color, background-color
  • font-size, font-weight
  • margin, padding, border
  • display: block, inline, flex, grid
  • position: static, relative, absolute, fixed, sticky
  • width, height, max-width
  • transition, transform, animation

JavaScript: Wichtige Konzepte

  • Variablen: let x = 5;, const name = "Test";
  • Funktionen: function hallo() { alert("Hallo!"); }
  • DOM: document.getElementById("id"), document.querySelector()
  • Events: element.addEventListener("click", fn)
  • Schleifen: for, while
  • Bedingungen: if, else, switch
  • Klassen & Objekte: class Auto {}
  • Arrays: [1,2,3].forEach(x => ...)
