Wichtige HTML-Objekte

  • <input type="text"> → Einfache Texteingabe
  • <input type="password"> → Passwortfeld
  • <input type="checkbox"> → Checkbox
  • <input type="radio"> → Radio-Button
  • <button> → Ein Button
  • <textarea> → Mehrzeiliges Eingabefeld
  • <select> <option> → Dropdown Auswahl
  • <img src="bild.jpg"> → Bild einfügen
  • <a href="...">Link</a> → Link
  • <video> / <audio> → Multimedia Elemente
